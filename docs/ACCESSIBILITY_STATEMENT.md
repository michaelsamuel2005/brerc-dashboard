## Accessibility statement for the BRERC species-distribution dashboard

This statement applies to the interactive species-distribution dashboard provided
by the Bristol Regional Environmental Records Centre (BRERC)[, embedded at
**[URL(s)]**].

This dashboard is run by **[BRERC / Bristol City Council]**. We want as many people
as possible to be able to use it. For example, that means you should be able to:

- navigate the dashboard and its filters using a keyboard;
- use a screen reader to read the species records as a data table;
- read text at 4.5:1 contrast and resize text up to 200% without loss of content.

[AbilityNet](https://mcmw.abilitynet.org.uk/) has advice on making your device
easier to use if you have a disability.

### How accessible this dashboard is

We believe the dashboard is **[compliant / partially compliant]** with the Web
Content Accessibility Guidelines version 2.2 AA standard. **[Describe any known
issues here, e.g. the interactive map itself.]**

The interactive map is an online map. Under the regulations, online maps are
exempt, **but** the essential information the map conveys — which species have been
recorded where, and how many records — is also available as an accessible,
filterable **data table** on the same page.
