# BRERC Dashboard

Team repository for the **180 Degrees Consulting Bristol × Bristol Regional
Environmental Records Centre (BRERC)** project — building a dashboard to present
the region's environmental and species-records data.

This is the team's shared workspace. The architecture, tools, folder structure,
and implementation are decided by the team as the project develops.

## Getting started

```bash
git clone https://github.com/michaelsamuel2005/brerc-dashboard.git
cd brerc-dashboard
```

Do your work on a branch and open a pull request for review — see
[CONTRIBUTING.md](CONTRIBUTING.md):

```bash
git switch -c <your-name>/<short-topic>
# ...your work...
git push -u origin <your-name>/<short-topic>
```
